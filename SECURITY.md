## Reporting a Vulnerability

We take the security of our software seriously. If you believe you have found a security vulnerability in the POS System, please report it to us as described below.

**Please do not report security vulnerabilities through public GitHub issues.**

### How to Report

Please send an email to [keshigami@gmail.com](mailto:keshigami@gmail.com) with the subject "Security Vulnerability".

Please include:
- A description of the vulnerability
- Steps to reproduce the issue
- Any relevant code snippets or screenshots

We will acknowledge your report within 48 hours and will work to verify and resolve the issue as quickly as possible.
